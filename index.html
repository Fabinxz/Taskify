Bem-vindo ao Taskify!

Este é um guia rápido para te ajudar a começar:

  • Contador Principal (no centro do cabeçalho): Use os botões + e − para adicionar ou remover questões resolvidas hoje. O número entre eles define quantas questões são adicionadas/removidas por clique.
  • Editar Metas: Clique no ícone no cabeçalho para definir suas metas diárias, semanais, mensais, anuais e de streak.
  • Alternar Tema: Use o ícone / no cabeçalho para trocar entre o tema escuro e claro.
  • Personalizar Aparência: Clique no ícone no cabeçalho para escolher paletas de cores e modos visuais.
  • Timer Pomodoro & Tarefas: Role a página para baixo para encontrar o Timer Pomodoro, a Lista de Tarefas e seus respectivos gráficos de atividade semanal.
  • Retrospectiva Mensal: Clique no ícone no canto inferior direito para ver sua retrospectiva do mês.
  • Resetar Dados: No modal de "Editar Metas" (), você encontrará um botão Resetar App para limpar todos os seus dados e começar do zero. Use com cuidado!
