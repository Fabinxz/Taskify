# Taskify 📊

Acompanhe seu progresso na resolução de questões com gráficos e metas personalizáveis.

![image](https://github.com/user-attachments/assets/95b0840c-ff60-4916-828b-7210a443c25d)

## Funcionalidades

- ▶️ Contador para questões resolvidas (Hoje/Semana/Mês/Ano)
- 🎯 Metas personalizáveis por período
- ⚡ Botões rápidos (+1, +5, +20, +50, -1)
- 🌗 Modo claro/escuro
- 🔄 Reset automático diário/semanal/mensal/anual
- 📊 Gráficos de pizza interativos

## Como Usar

1. **Adicionar questões**: Clique nos botões (+1, +5, +20, +50)
2. **Remover questões**: Use o botão (-1) 
3. **Editar metas**: 
   - Clique no ícone de lápis (✏️)
   - Ajuste os valores desejados
   - Salve as novas metas
